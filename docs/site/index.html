Quick Start

1. Install

npm install @aigraphia/protocol @aigraphia/plugins @aigraphia/cli

2. Create a Diagram

{
  "schema": "https://aigraphia.com/schema/v1",
  "version": "1.0.0",
  "type": "flowchart",
  "metadata": {
    "title": "My First Diagram"
  },
  "graph": {
    "nodes": [
      { "id": "start", "type": "start", "label": "Begin" },
      { "id": "end", "type": "end", "label": "End" }
    ],
    "edges": [
      { "id": "e1", "source": "start", "target": "end" }
    ]
  }
}

3. Validate & Convert

aigp validate diagram.json
aigp convert diagram.json mermaid

  • AI Agent Communication - Standard format for Claude, ChatGPT, and other LLMs to create and edit diagrams

  • Visual Programming - Build visual editors with a universal backend format
  • Workflow Automation - Define and execute workflows in a machine-readable format
  • Data Visualization - Create complex data flow and relationship visualizations
  • Knowledge Management - Structured representation of knowledge graphs and mind maps
